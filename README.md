# Dès
## Lancer de dès aléatoire

### Cloner le projet
```sh
git clone https://github.com/avencat/des.git
```

### Compiler le projet
```sh
cd des
make
```

### Effacer les fichiers temporaires
```sh
make clean
```

### Nettoyer le dossier des fichiers temporaires et de l'exécutable
```sh
make fclean
```

### Lancer les dès
```sh
./des
```
